智能展厅布局设计系统

系统概述

基于Qt框架和OpenCV开发的智能展厅布局设计软件，集成了轮廓识别、自动布局和科学评价三大核心功能，为展厅设计提供全方位的智能化解决方案。

开发环境
开发框架: Qt 5.15 或 Qt 6.0 及以上

依赖库: OpenCV

可选组件: Python 3.8（用于评价指标计算）

目标系统: Windows 10/11

核心功能模块

一级模块1：智能轮廓识别

该模块是整个布局设计流程的基石，负责将原始图纸转化为精确的数字空间模型。它通过先进的图像识别与CAD解析技术，实现空间边界自动提取。为确保万无一失，系统还配备了强大的轮廓手动修正工具，允许设计师对自动识别的结果进行精细化调整。同时，内置的障碍物标记系统能够方便地标出墙壁等固定障碍物，为后续的智能布局提供完整、准确的空间约束条件，从源头上保障了设计的可行性。

一级模块2：自动布局设计

此模块是软件的智能核心，旨在将设计师从繁琐的重复性摆放工作中解放出来。它内置了丰富的展具模型库与行业领先的布局算法，能够根据用户需求执行陈列物件智能摆放，综合考虑展品类型、尺寸与关联性。其核心的空间利用率优化算法能在保证通道畅通和观展舒适度的前提下，最大化利用有效展示面积。此外，该模块支持多方案对比生成，可快速产出多种不同风格和侧重点的布局方案，极大地拓宽了设计思路，为用户提供了丰富的选择空间。

一级模块3：科学评价分析

为解决设计方案优劣难以量化的痛点，该模块构建了一套全面、客观的评价体系。它从美学指标评估（如整齐规范、对称均衡、节奏韵律）出发，对方案的艺术性进行打分；同时，通过功能指标检测（如可视可达性、通道宽度），量化评估布局的实用性与观众体验。更重要的是，系统内置了安全标准校验功能，能够自动对照国家公共安全规范进行审查，确保设计方案在美观与实用之外，完全符合安全要求，为方案的最终落地提供了科学、可靠的决策依据。
